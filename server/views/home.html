Hello freeCodeCamp challenge!

To get a list of the most recently submitted search strings, please try

  • /api/latest/imagesearch: default limit is 10
  • /api/latest/imagesearch?limit=20
  • /api/latest/imagesearch?limit=51: max limit is 50

To get the image URLs, alt text and page urls for a set of images relating to a given search string, please try

  • /api/imagesearch/github%20huynhsamha: default offset is 0, limit is 10
  • /api/imagesearch/github%20huynhsamha?offset=5
  • /api/imagesearch/huynhsamha?offset=0&limit=15: if query has offset, that will make offset is 1
  • /api/imagesearch/github?offset=20&limit=20: max limit is 10
